# 💎 Adore Jewelry Website

Modern jewelry web application with elegant UI and responsive design.

## ✨ Features
- Elegant and modern jewelry UI
- Responsive website design
- Product showcase section
- Attractive homepage layout
- Admin panel support
- Smooth navigation experience

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Flask
- Python
